```
docker rename 命令用于更改已存在的 Docker 容器或镜像的名称。可以更好地管理和组织你的容器或镜像

✅ 基本语法
docker rename OLD_NAME NEW_NAME


✅ 示例
重命名容器
docker rename my_old_container my_new_container

重命名镜像
docker rename my_old_image my_new_image


✅ 注意事项
唯一性：新名称必须是唯一的，不能与现有的容器或镜像名称冲突。
运行状态：可以重命名正在运行的容器，无需停止它。
不会影响功能：重命名操作不会影响容器或镜像的功能或数据。
```
